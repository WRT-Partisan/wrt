物质的组成、分类及转化

根据物质的组成和性质分类

分类

分类的意义：更方便认识和研究物质，能解释和预测物质的性质，更容易寻找化学反应的规律。

常见的分类方法：

  1. 树状分类法：对同类物质按照某些属性进行再分类的分类法，其特点是能认识各类物质之间的从属关系。
  2. 交叉分类法：按不同标准对同一物质进行多次分类的分类方法，其特点是能从多个角度认识某一物质的多重属性。

同素异形体

定义：由同种元素形成的不同性质的单质。

如：金刚石、石墨、\ce{C60}；红磷、白磷、黑鳞、绿磷；氧气、臭氧；白锡、灰锡。

注意：

  1. 根据“结构决定性质”可知，同素异形体的性质不同，是因为单质分子的结构不同。
  2. 同素异形体相互转化属于化学变化。
  3. 只含一种元素的物质不一定是纯净物。

氧化物

定义：由两种元素组成且其中一种元素为 \ce{O} 的化合物。

氧化物分为成盐氧化物（即有相应价态的盐）和不成盐氧化物（即没有相应价态的盐）。其中成盐氧化物又分为酸性氧化物、碱性氧化物、两性氧化物。

不成盐氧化物：如 \ce{CO}、\ce{NO}。

酸性氧化物

定义：能与碱反应生成盐和水，且化合价不改变的氧化物。

常见物质种类：绝大多数非金属氧化物（如 \ce{CO2}、\ce{SO2}、\ce{SO3}、\ce{P2O5}、\ce{SiO2}）、部分金属氧化物（如 \ce{Mn2O7}）。

通性：

  1. 与碱反应生成盐和水，且化合价不变。如 \ce{Mn2O7 + 2KOH = 2KMnO4 + H2O}。
  2. 与碱性氧化物反应生成盐，且化合价不变。如 \ce{CO2 + Na2O = Na2CO3}。
  3. 若能与水反应，则生成对应价态的酸。如 \ce{\overset{+4}{C}O2 + H2O = H2\overset{+4}{C}O3}，\ce{\overset{+4}{S}O2 -> H2\overset{+4}{S}O3}，\ce{\overset{+6}{S}O3 -> H2\overset{+6}{S}O4}。

注：

  1. 若氧化物存在等化合价的酸根，则该氧化物为酸性氧化物，如 \ce{CO2}、\ce{SO2}、\ce{SO3}、\ce{N2O5}、\ce{P2O5}、\ce{SiO2} 等；\ce{NO2} 不存在 +4+4+4 价 \ce{N} 对应的酸根，故 \ce{NO2} 不是酸性氧化物（\ce{3NO2 + H2O = 2HNO3 + NO}，\ce{2NO2 + 2NaOH = NaNO3 + NaNO2 + H2O}）。
  2. 不是所有的酸性氧化物都能与水反应生成对应价态的酸，如 \ce{SiO2}（难溶于水）就不能与水反应生成 \ce{H2SiO3}；只有能溶于水的酸性氧化物才能与水反应生成对应价态的酸，如 \ce{CO2}、\ce{SO2}、\ce{SO3} 等。

碱性氧化物

定义：能与酸反应生成盐和水，且化合价不改变的氧化物。

常见的物质种类：绝大多数金属氧化物。例外：\ce{Mn2O7}（酸性氧化物）、\ce{Al2O3}（两性氧化物）。

通性：

  1. 与酸反应生成盐和水，且化合价不变。如 \ce{\overset{+1}{Na}_2O + H2SO4 = \overset{+1}{Na}_2SO4 + H2O}。
  2. 与酸性氧化物反应生成盐，且化合价不变。
  3. 若能与水反应，则生成对应价态的碱。如 \ce{CaO + H2O = Ca(OH)2}、\ce{Na2O + H2O = 2NaOH}。

注：

  1. 过氧化物如 \ce{Na2O2} 一定不是碱性氧化物，因为其中的氧元素化合价为 −1-1−1 价，与酸反应时化合价会改变。\ce{2Na2\overset{-1}{O}_2 + 4HCl = 4NaCl + 2H2\overset{-2}{O} + \overset{0}{O}_2 ^}。
  2. 不是所有碱性氧化物都能与水反应生成对应价态的碱，如 \ce{Fe2O3}、CuOCuOCuO 等就不能与水反应生成 \ce{Fe(OH)3}、\ce{Cu(OH)2}，只有能溶于水的碱性氧化物才能与水反应生成对应价态的碱，如 \ce{CaO}、\ce{K2O}、\ce{Na2O}、\ce{BaO} 等。
  3. 可以通过类比法和对比法，参照酸性氧化物来学习碱性氧化物。

两性氧化物

定义：既能与酸反应转化为盐，又能与碱反应转化为盐，且化合价均不改变的氧化物。\ce{\overset{+3}{Al}_2O3 + 6HCl = 2\overset{+3}{Al}Cl3 + 3H2O}，\ce{\overset{+3}{Al}_2O3 + 2NaOH + 3H2O = 2Na[\overset{+3}{Al}(OH)4]}（四羟基合铝酸钠）。

常见物质种类：\ce{Al2O3}、\ce{ZnO}、\ce{Ga2O3}、\ce{BeO} 等。

通性：酸性氧化物和碱性氧化物通性的总和。

注：

  1. \ce{SiO2} 既能与氢氟酸（\ce{HF}）反应，又能与碱反应，但 \ce{SiO2} 不是两性氧化物，因为 \ce{SiO2} 与 \ce{HF} 反应的产物 SiF4SiF4SiF4 不是盐（盐是能电离出金属阳离子或铵根离子和酸根离子的化合物）。\ce{SiO2 + 4HF = SiF4 ^ + 2H2O}，\ce{SiO2 + 2NaOH = Na2SiO3 + H2O}。SiO2SiO2SiO2 是酸性氧化物。
  2. 氧化物中除去酸性氧化物、碱性氧化物和两性氧化物之外，还有不成盐氧化物、过氧化物等。

分散系及其分类

分散系

定义:一种（或多种）物质以粒子形式分散到另一种（或多种）物质中所形成的混合物。

组成：分散质和分散剂（一般将量少的视为分散质，如果有水一般将水视为分散剂）。

分散系的分类

根据分散质、分散剂的状态将分散系分为 999 类（一般先描述分散质，再描述分散剂）。

  1. 气气分散系：分散质是气态，分散剂是气态。
  2. 气液分散系：分散质是气态，分散剂是液态。
  3. 气固分散系：分散质是气态，分散剂是固态。
  4. 液气分散系：分散质是液态，分散剂是气态。
  5. 液液分散系：分散质是液态，分散剂是液态。
  6. 液固分散系：分散质是液态，分散剂是固态。
  7. 固气分散系：分散质是固态，分散剂是气态。
  8. 固液分散系：分散质是固态，分散剂是液态。
  9. 固固分散系：分散质是固态，分散剂是固态。

根据分散质粒子的直径大小将分散系分为 333 类。

  1. 溶液：分散质粒子直径小于 1nm1\text{nm}1nm。
  2. 胶体：分散质粒子直径在 1nm1\text{nm}1nm 到 100nm100\text{nm}100nm 之间。
  3. 浊液：分散质粒子直径大于 100nm100\text{nm}100nm。浊液又根据分散质的状态分为乳浊液（分散质为液态）和悬浊液（分散质为固态）。

注：

  1. 混合物不一定是分散系（如分层的油和水），分散系一定是混合物（如振荡后的油水混合物形成的乳浊液）。
  2. 胶体属于分散系，所以胶体肯定是混合物。粒子直径为 50nm50\text{nm}50nm 的 \ce{CaCO3} 粉末不是胶体，因为 \ce{CaCO3} 是纯净物。将粒子直径为 50nm50\text{nm}50nm 的 \ce{CaCO3} 粉末分散到空气中或水中形成的分散系才称为胶体。

胶体

定义：分散质粒子直径在 1nm1\text{nm}1nm 到 100nm100\text{nm}100nm 之间的分散系。

本质特征：结构决定性质，胶体区别于其他分散系的本质特征是分散质粒子直径在 1nm1\text{nm}1nm 到 100nm100\text{nm}100nm 之间（1m=102cm=103mm=106μm=109nm=1012pm1\text{m}=10^2\text{cm}=10^3\text{mm}=10^6\mu\text{m}=10^9\text{nm}=10^{12}\text{pm}1m=102cm=103mm=106μm=109nm=1012pm）。

分类：根据分散剂状态不同，可将胶体分为 333 类，分别是气溶胶（如雾）、液溶胶（如 \ce{Fe(OH)3} 胶体）和固溶胶（如有色玻璃）。

胶体的性质及其用途
丁达尔效应

定义：当光束通过胶体时，可以看到一条光亮的“通路”，这种现象称为丁达尔效应。

原因：胶体粒子使光波发生散射（光波偏离原来的方向而分散传播）。

运用：区别胶体和溶液。

电泳

定义：通电时，带电的胶体粒子定向移动，这种现象称为电泳。

原因：胶体分散质具有巨大的比表面积，能吸附带电荷的离子使胶体粒子带电。

运用：电泳电镀、静电除尘、蛋白质分离等。

注：

  1. 胶体不带电，胶体粒子才可能带电；不是所有胶体粒子都带电，例如淀粉胶体粒子不带电。
  2. 一般情况下，难溶性碱形成的胶体粒子带正电，如 \ce{Fe(OH)3} 胶体粒子；土壤胶体粒子一般带负电，能与 \ce{K+}、\ce{NH4+} 等生物所需营养离子相互作用，使土壤具有保肥能力。
  3. 比表面积是指单位质量的物质所具有的总面积。
聚沉

定义：胶体的分散质粒子聚集成较大的微粒，形成沉淀从分散系里析出的现象。

措施：加热、搅拌、加入电解质或加入带相反电荷胶体粒子的胶体。

运用：氯化铁止血、石膏（或氯化镁）溶液使豆浆变成豆腐、入海口形成三角洲、明矾净水、墨水混用导致笔管堵塞等。

注：往氢氧化铁胶体中滴加稀硫酸，先生成红褐色沉淀，继续滴加稀硫酸后沉淀溶解，溶液变为棕黄色。上述现象说明，向胶体中加入少量电解质时发生聚沉，当电解质过量时可能发生化学反应。

渗析

定义：利用半透膜分离溶液和胶体的过程。

原因：溶液分散质粒子的直径小于半透膜孔径，而胶体粒子的直径大于半透膜孔径。

运用：肾小球的滤过作用、血液透析、淀粉与葡萄糖的分离等。

注：不能用滤纸过滤、分离和提纯胶体，因为胶体粒子和溶液的分散质粒子都可以穿过滤纸。

胶体的制备

操作：将 555 ~ 666 滴 \ce{FeCl3} 饱和溶液滴加到 40mL40\text{mL}40mL 煮沸的蒸馏水中，继续煮沸至液体呈红褐色，停止加热。

原理：\ce{FeCl3 + 3H2O ->[\Delta] Fe(OH)3(\text{胶体}) + 3HCl}。

注意事项：

  1. 蒸馏水不能用自来水、氨水、氢氧化钠溶液等替代，否则形成 \ce{Fe(OH)3} 沉淀而无法获得胶体。
  2. 不能长时间煮沸，不能搅拌。

酸、碱、盐的性质

酸

定义：电离出的阳离子全部都是 \ce{H+} 的化合物。

分类：

  1. 根据酸分子是否含氧原子分为含氧酸（如 \ce{CH3COOH}、\ce{HClO}）和无氧酸（如 \ce{HCl}、\ce{HBr}）。
  2. 根据每个分子可电离出的 \ce{H+} 个数分为一元酸（如 \ce{CH3COOH}、\ce{HCl}）、二元酸（如 \ce{H2SO4}、\ce{H2CO3}）、多元酸（如 \ce{H3PO4}）。
  3. 根据酸的强弱分为强酸（\ce{HCl}、\ce{HBr}、\ce{HI}、\ce{H2SO4}、\ce{HNO3}、\ce{HClO4}，为六大强酸）、弱酸（\ce{CH3COOH}、\ce{HClO}、\ce{H2CO3} 等）。

注：酸的“元数”不是看分子中氢原子的个数，而是看酸分子可电离出的氢离子个数。如 \ce{H3PO4}、\ce{H3PO3}、\ce{H3PO2} 分子中都有 333 个氢原子，但每一个酸分子可电离出的氢离子数分别是 333、222、111，因此 \ce{H3PO4} 为三元酸，\ce{H3PO3} 为二元酸，\ce{H3PO2} 为一元酸。

酸的通性：

  1. 酸能使指示剂显色（使紫色石蕊溶液变红）。
  2. \ce{\text{酸} + \text{碱} -> \text{盐} + H2O}。
  3. \ce{\text{酸} + \text{碱性氧化物} -> \text{盐} + H2O}。
  4. \ce{\text{酸} + \text{盐} -> \text{新酸} + \text{新盐}}。
  5. \ce{\text{酸（非氧化性酸）} + \text{活泼金属单质} -> \text{盐} + H2 ^}。

注：\ce{\text{酸} + \text{盐} -> \text{新酸} + \text{新盐}}，该反应过程一般遵循“强酸制弱酸”或“溶解度大的盐制溶解度小的盐”。此原理中的强与弱、大与小都是相对的，而非绝对。弱酸也可以制更弱的酸，如 \ce{CH3COOH + NaClO = CH3COONa + HClO}，说明 \ce{CH3COOH} 的酸性比 \ce{HClO} 的强。

碱

定义：电离出的阴离子全部都是 \ce{OH-} 的化合物。

分类：

  1. 根据碱在水中的溶解度分为易溶性碱（如 \ce{NaOH}、\ce{KOH}、\ce{Ba(OH)2}）、微溶性碱（如 \ce{Ca(OH)2}）和难溶性碱（如 \ce{Mg(OH)2}、\ce{Cu(OH)2}）。四大微溶物质：\ce{CaSO4}、\ce{MgCO3}、\ce{Ag2SO4}、\ce{Ca(OH)2}。
  2. 根据每个碱可电离出的 \ce{OH-} 个数分为一元碱（如 \ce{NH3.H2O}、\ce{NaOH}）、二元碱（如 \ce{Ba(OH)2}、\ce{Ca(OH)2} 等）。
  3. 根据碱的强弱分为强碱（\ce{Ba(OH)2}、\ce{Ca(OH)2}、\ce{NaOH}、\ce{KOH}）、弱碱（\ce{NH3.H2O}、\ce{Mg(OH)2}、\ce{Cu(OH)2} 等）

通性：

  1. 碱能使指示剂显色（使紫色石蕊溶液变蓝，使酚酞溶液变红）。
  2. \ce{\text{碱} + \text{酸} -> \text{盐} + H2O}。
  3. \ce{\text{碱} + \text{酸性氧化物} -> \text{盐} + H2O}。
  4. \ce{\text{碱} + \text{盐} -> \text{新碱} + \text{新盐}}。

注：\ce{NH3.H2O} 作为产物，若体系温度高写成 \ce{NH3 ^ + H2O}；若体系温度低写成 \ce{NH3.H2O}。

盐

定义：能电离出金属阳离子（或铵根离子）和酸根离子的化合物。

分类：

  1. 根据电离成分分为正盐（如 \ce{BaCl2}）、酸式盐（如 \ce{NaHSO4}）、碱式盐（如 \ce{Cu2(OH)2CO3}）、复盐（如 \ce{KAl(SO4)2.12H2O}）等。
  2. 根据溶解度分为易溶性盐（如绝大多数钾、钠、铵盐）、微溶性盐（如 \ce{CaSO4}、\ce{MgCO3}、\ce{Ag2SO4}）和难溶性盐（如 \ce{BaSO4}）。

注：

  1. 正盐是指只能电离出金属阳离子（或铵根离子）和酸根离子的盐，不能电离出 \ce{H+} 和 \ce{OH-}。
  2. 能电离出 \ce{H+} 的盐称为酸式盐，能电离出 \ce{OH-} 的盐称为碱式盐。酸式盐的水溶液不一定呈酸性，如酸式盐 \ce{NaHSO4}、\ce{NaHCO3} 的溶液分别呈酸性、碱性。酸式盐、碱式盐是盐从电离成分来分类的结果，不是按照水溶液的酸碱性来分类。
  3. 能电离出的阳离子（除 \ce{H+} 外）大于等于两种的盐称为复盐，如 \ce{NH4Fe(SO4)2}。

通性：

  1. \ce{\text{盐} + \text{酸} -> \text{新盐} + \text{新酸}}。
  2. \ce{\text{盐} + \text{碱} -> \text{新盐} + \text{新碱}}。
  3. \ce{\text{盐} + \text{盐} -> \text{新盐} + \text{新盐}}。
  4. \ce{\text{盐} + \text{金属单质} -> \text{新盐} + \text{新金属单质}}。

物质的转化

四大基本反应类型及其通式

反应类型 反应通式 实例
化合反应 \ce{A + B = AB} \ce{H2 + Cl2 ->T[点燃]} 2HCl
分解反应 \ce{AB = A + B} \ce{2HClO ->T[光照] 2HCl + O2 ^}
置换反应 \ce{A + BC = AB + C} \ce{Cl2 + 2KI = 2KCl + I2}
复分解反应 \ce{AB + CD = AD + BC} \ce{CuCl2 + H2S = CuS v + 2HCl}

注：四大基本反应不能包括所有化学反应，如 \ce{CO + CuO ->[\Delta] Cu + CO2} 不属于四大基本反应类型中的任何一类。

常见物质的分类

混合物

  • 煤：含有碳、氢、氧、氮、硫、（磷）、（氯）、（砷）等元素。
  • 石油：主要成分为碳氢化合物（烷烃、环烷烃、芳香烃），同时还含有少量的硫、氮、氧等元素。
  • 天然气：主要成分为甲烷。
  • 液化石油气：主要成分为丙烷和丁烷。
  • 水煤气：成分为一氧化碳和氢气。\ce{C + H2O(g) ->T[高温] CO2 + H2}。
  • 可燃冰：主要成分为被水分子包围的甲烷。

有机物

  • 有机高分子化合物（聚合物）：淀粉（\ce{(C6H10O5)_n}）、纤维素（\ce{(C6H10O5)_n}）、蛋白质、塑料、橡胶、合成纤维、树脂。
  • 油脂：一个油脂分子由一个甘油和三个脂肪酸构成。
  • 福尔马林：35%35\%35% 到 40%40\%40% 的甲醛水溶液。
  • 冰醋酸：默认为纯净物。

不同物态的纯净物

  • 干冰：固态 \ce{CO2}。
  • 冰水混合物：介于固态和液态之间的 \ce{H2O}。
  • 液氮：液态的 \ce{N2}。
  • 液氨：液态的 \ce{NH3}。
  • 液氯：液态的 \ce{Cl2}。

常见水溶液

  • 硫酸：既可以指 \ce{H2SO4}，也可以指 \ce{H2SO4} 的水溶液。考试时默认为纯净物，若给出浓度则为水溶液。
  • 硝酸：既可以指 \ce{HNO3}，也可以指 \ce{HNO3} 的水溶液。考试时默认为纯净物，若给出浓度则为水溶液。
  • 盐酸：\ce{HCl} 气体的水溶液，是混合物。
  • 氯水：将 \ce{Cl2} 通入水中形成的产物，含有 \ce{Cl2}、\ce{HCl}、\ce{HClO}、\ce{H2O}。\ce{Cl2 + H2O <=> HCl + HClO}。
  • 氨水：将 \ce{NH3} 通入水中形成的产物，或 \ce{NH3.H2O} 的水溶液。
  • 双氧水：\ce{H2O2} 的水溶液。
  • 王水：浓 \ce{HNO3} 和浓 \ce{HCl} 按照 1:31:31:3 的比例混合，具有很强的腐蚀性。
  • 84 消毒液/漂白液：\ce{Cl2 + 2NaOH = NaClO + NaCl + H2O}，含有 \ce{NaClO}、\ce{NaCl}、\ce{H2O}。
  • 漂白粉：\ce{2Cl2 + 2Ca(OH)2 = Ca(ClO)2 + CaCl2 + 2H2O}，含有 \ce{Ca(ClO)2}、\ce{CaCl2}、\ce{H2O}。

常见固体

  • 玻璃：为混合物，主要成分为 \ce{SiO2}。
  • 水玻璃：\ce{Na2SiO3} 的水溶液。
  • 碱石灰：\ce{CaO}、\ce{NaOH}、\ce{KOH} 的混合物，常用作干燥剂。
  • 钢：为混合物，主要成分为铁，还有少量碳。
  • 合金（如青铜、黄铜）：为混合物，由多种金属（或多种物质其中至少有一种金属）混合而成。
  • 铝热剂：为混合物，有很多种类。其中一种为 \ce{Al} 和 \ce{Fe2O3} 的混合物。

其他易混淆物质

  • 胆矾/蓝矾：五水合硫酸铜（\ce{CuSO4.5H2O}），为纯净物。
  • 明矾：十二水合硫酸铝钾（\ce{KAl(SO4)2.12H2O}），为纯净物。
  • 纯净矿泉水：为混合物，有大量电解质。
  • 纯净无污染空气：为混合物，主要成分为 \ce{N2}、\ce{O2}。
  • 澄清石灰水：\ce{Ca(OH)2} 的水溶液，为混合物。

离子反应

电解质与电离方程式

物质导电的本质及影响因素

金属导电：

  1. 本质：金属的自由电子在外加电场下的定向移动。
  2. 影响因素：金属的电阻率、导线的长度、横截面积、温度等。

溶液导电：

  1. 本质：自由移动的离子在外加电场下的定向移动。
  2. 影响因素：电荷浓度=离子浓度×离子所带电荷数\text{电荷浓度}=\text{离子浓度}\times\text{离子所带电荷数}电荷浓度=离子浓度×离子所带电荷数。

注：

  1. \ce{NaCl} 固体由 \ce{Na+}、\ce{Cl-} 构成，但这些离子不能自由移动，因此 \ce{NaCl} 固体不导电。
  2. 影响电解质溶液导电性的因素，还有溶液温度、离子迁移速率等。在高中阶段，大多数题目主要考虑的影响因素是电荷浓度。

电解质及其分类

定义：在水溶液里或熔融状态下能够导电的化合物。

理解：

  1. “或”表示满足其一即可。如 \ce{HCl} 的液态（即熔融态）不导电，但 \ce{HCl} 溶于水形成的盐酸能导电，则 \ce{HCl} 是电解质；\ce{BaSO4} 难溶于水，其水溶液几乎不导电，但将其熔化，其熔融态能导电，则 \ce{BaSO4} 是电解质。\ce{NaCl} 的水溶液以及熔融状态都能导电，所以 \ce{NaCl} 也是电解质。
  2. “化合物”说明电解质必然是纯净物，单质和溶液既不是电解质，也不是非电解质。
  3. \ce{NH3}、\ce{SO2} 的水溶液可导电，但 \ce{NH3}、\ce{SO2} 却不是电解质，这是因为 \ce{NH3}、\ce{SO2} 溶于水，可与水发生反应：\ce{NH3 + H2O <=> NH3.H2O}、\ce{SO2 + H2O <=> H2SO3}。溶液中自由移动的离子来自 \ce{NH3.H2O}、\ce{H2SO3}，而非 \ce{NH3}、\ce{SO2}，\ce{NH3.H2O}、\ce{H2SO3} 是电解质。

常见的电解质：酸、碱、盐、\ce{H2O}（\ce{H2O <=> H+ + OH-}）和活泼金属氧化物。

分类：根据电解质在水中的电离程度，电解质分为强电解质和弱电解质。若完全电离，则为强电解质；若部分电离，则为弱电解质。

强电解质：

  • 活泼金属氧化物：\ce{Na2O}、\ce{CaO}、\ce{BaO} 等。
  • 强酸：\ce{HNO3}、\ce{H2SO4}、\ce{HClO4}、\ce{HCl}、\ce{HBr}、\ce{HI} 等。
  • 强碱：\ce{NaOH}、\ce{KOH}、\ce{Ca(OH)2}、\ce{Ba(OH)2} 等。
  • 绝大多数的盐：\ce{NaCl}、\ce{KNO3}、\ce{BaSO4} 等。

弱电解质：

  • 弱酸：\ce{CH3COOH}、\ce{H2CO3}、\ce{HClO}、\ce{H3BO3} 等。
  • 弱碱：\ce{NH3.H2O}、\ce{Mg(OH)2}、\ce{Fe(OH)2}、\ce{Fe(OH)3} 等。
  • 极少数盐和水：\ce{(CH3COO)2Pb}、\ce{Fe(SCN)3}、\ce{H2O} 等。

电解质的电离与导电原理

电离的定义：电解质溶于水或受热熔化时，形成自由移动的离子的过程

电解质在水中的电离（以 \ce{NaCl} 为例）：当将 \ce{NaCl} 固体加入水中时，在水分子的作用下，\ce{Na+} 和 \ce{Cl-} 脱离 \ce{NaCl} 固体的表面，进入水中，形成能够自由移动的水合钠离子和水合氯离子。

注：水分子的一个氧原子偏负电性，水分子中的两个氢原子偏正电性，故水合钠离子中水分子的氧原子偏向钠离子，水合氯离子中水分子的氢原子偏向氯离子。

电解质水溶液的导电原理（以 \ce{NaCl} 为例）：当在 \ce{NaCl} 溶液中插入电极并接通电源时，带正电荷的水合钠离子向与电源负极相连的电极移动，带负电荷的水合氯离子向与电源正极相连的电极移动，因而 \ce{NaCl} 溶液能够导电。

电解质熔融态的导电原理（以 \ce{NaCl} 为例）：当 \ce{NaCl} 固体受热熔化时，离子的运动随温度升高而加快，克服了离子间的相互作用，产生了能够自由移动的 \ce{Na+} 和 \ce{Cl-}，因而 \ce{NaCl} 在熔融状态时也能导电。

注：

  1. 电离是物理变化，电解是化学变化。
  2. 电离的条件是溶解或熔化，电解的条件是通电（\ce{2H2O ->T[通电] 2H2 ^ + O2 ^}）。
  3. 电离示意图中，一定要体现“异性相吸”的规律。

电离方程式

定义

表示电解质电离过程的式子。

书写方法

左侧为电解质，右侧为离子，连接符号用“\ce{=}”或“\ce{<=>}”。强电解质的电离过程用“\ce{=}”连接，弱电解质的电离过程用“\ce{<=>}”连接。

如：

\ce{H2SO4 = 2H+ + SO4^{2-}}

\ce{Ba(OH)2=Ba^{2+} + 2OH-}

\ce{Na2CO3 = 2Na+ + CO3^{2-}}

\ce{HF <=> H+ + F-}

绝大多数的多元弱酸酸式酸根（如 \ce{HSO3-}、\ce{HCO3-} 等）在水中不能完全电离，但 \ce{HSO4-} 在水溶液中能完全电离，则 \ce{NaHSO4} 在水中电离方程式为 \ce{NaHSO4 = Na+ + H+ + SO4^{2-}}。\ce{HSO4-} 在熔融状态下不能电离，则 \ce{NaHSO4} 在熔融状态下的电离方程式为 \ce{NaHSO4 = Na+ + HSO4-}。

书写电离方程式需要配平，其遵循“原子守恒”和“电荷守恒”。“电荷守恒”是指电离方程式左右两边总电荷的数值和符号完全相同。

注：

  1. 多元弱酸分步电离，以第一步为主。如 \ce{H2CO3} 的电离方程式为 \ce{H2CO3 <=> H+ + HCO3-}。
  2. 多元弱碱实质分步电离，为简便起见，视为全部电离，但连接符号仍用“\ce{<=>}”。如 \ce{Al(OH)3} 的电离方程式为 \ce{Al(OH)3 <=> Al^{3+} + 3OH-}。

离子反应及其表示方法

离子反应：有离子参加或生成的反应。

离子方程式：用实际参加反应的离子符号来表示反应的式子。

注：电离方程式表示电离（物理变化）的过程，离子方程式表示化学反应的过程。

离子方程式地方书写步骤（以 \ce{AgNO3} 与 \ce{NaCl} 反应为例）：

  1. 写：写出反应的化学方程式：\ce{AgNO3 + NaCl = AgCl v + NaNO3}。
  2. 拆：把易溶于水且易电离的物质（如强酸、强碱和大部分可溶性盐）写成离子形式，难溶的物质、气体和水等仍用化学式表示。\ce{Ag+ + NO3- + Na+ + Cl- = AgCl v + Na+ + NO3-}。
  3. 删：删去方程式两边不参加反应的离子，并将方程式化为最简：\ce{Ag+ + Cl- = AgCl v}。
  4. 查：检查离子方程式两边各元素的原子个数和电荷总数是否相等。

注：

  1. 在水溶液中，完全以自由移动的离子形式存在的物质才能拆成离子的形式。
  2. 强酸中浓硫酸不拆，因为其体系中的水很少，而 \ce{H2SO4} 分子要依赖水才能电离。浓硫酸体系中的 \ce{H2SO4} 未完全电离，所以不拆。
  3. 微溶物作为产物一律不拆；作为反应物，浑浊不拆，澄清则拆。
  4. 易溶的强电解质盐，若作为产物以沉淀的形式析出，则不拆。如侯氏制碱法反应中 \ce{NaCl + H2O + NH3 + CO2 = NH4Cl + NaHCO3 v}，其离子方程式为 \ce{Na+ + H2O + NH3 + CO2 = NH4+ + NaHCO3 v}。
  5. 氧化物一律不拆。

离子方程式的意义：离子方程式不仅可以表示某个具体的化学反应，还可以表示同一类型的离子反应。

复分解型离子反应发生的条件：

  1. 溶解度大制溶解度小。
  2. 难挥发（沸点高）制易挥发（沸点低）。
  3. 强电解质制弱电解质。

注：上述三个条件满足其一即可，部分反应甚至可能违背上述规律。如 \ce{H2S + CuSO4 = CuS v + H2SO4} 违背了强酸制弱酸，但遵循溶解度大制溶解度小，故该反应可以进行。

离子共存

离子共存的本质

离子共存的本质：体系中的几种离子不发生化学反应，即各个离子不生成沉淀、气体、弱电解质。

注：

  1. 离子反应除了复分解型离子反应，还有氧化还原反应型离子反应和配位型离子反应。
  2. 离子共存默认是大量离子共存，比如大量 \ce{H+} 和 \ce{OH-} 虽然会发生反应，但水中却可以同时存在少量 \ce{H+} 和 \ce{OH-}。

离子大量共存的附加条件

常见有色离子：

  1. \ce{Cr2O7^{2-}}（重铬酸根离子）：橙色。
  2. \ce{Fe^{3+}}、\ce{CrO4^{2-}}（铬酸根离子）：黄色。
  3. \ce{Fe^{2+}}、\ce{Cr^{3+}}：绿色。
  4. \ce{Cu^{2+}}：蓝色。
  5. \ce{MnO4-}：紫红色。

注：

  1. “透明”是指没有沉淀的澄清溶液，有颜色的溶液可以澄清透明，因此“透明”和“有色”不冲突。如无色溶液中，\ce{Cu^{2+}} 不能大量存在；但透明溶液中，\ce{Cu^{2+}} 可以大量存在。
  2. 记忆离子颜色时，按照红橙黄绿蓝靛紫的顺序进行记忆，更易记牢。

溶液酸碱性：

  1. 指示剂显色：酚酞变红或石蕊变蓝为碱性，石蕊变红为酸性。
  2. 常温下溶液 \ce{pH}：\ce{pH}<7 为酸性，\ce{pH}>7 为碱性。

常见反应组合（不能共存的组合）

因生成弱电解质而不能大量共存

\ce{H+}：

  1. 弱酸阴离子：\ce{CO3^{2-}}、\ce{HCO3-}、\ce{SO3^{2-}}、\ce{HSO3-}、\ce{S^{2-}}、\ce{HS-}、\ce{PO4^{3-}}、\ce{H2PO4-}、\ce{HPO4^{2-}}、\ce{ClO-}、\ce{F-} 等，与 \ce{H+} 反应生成弱酸而不能大量共存。
  2. \ce{[Al(OH)4]-} 与 \ce{H+} 反应生成 \ce{Al(OH)3} 或 \ce{Al^{3+}}，二者无法大量共存。
  3. \ce{H+} 和 \ce{OH-} 不能大量共存。

\ce{OH-}：

  1. 弱碱阳离子（\ce{Mg^{2+}}、\ce{Al^{3+}}、\ce{Fe^{3+}}、\ce{Fe^{2+}}、\ce{Cu^{2+}}、\ce{NH4+} 等）与 \ce{OH-} 反应生成弱碱而不能大量共存。
  2. 弱酸的酸式酸根（\ce{HCO3-}、\ce{HSO3-}、\ce{HS-}、\ce{HaPO4-}、\ce{HPO4^{2-}} 等）与 \ce{OH-} 反应生成水和酸根离子而不能大量共存。
  3. \ce{H+} 和 \ce{OH-} 不能大量共存。
因生成沉淀（或微溶性物质）而不能大量共存
饮离子 不能大量共存的阳离子
\ce{Cl-}、\ce{Br-}、\ce{I-} \ce{Ag+}
\ce{SO4^{2-}} \ce{Ba^{2+}}、\ce{Pb^{2+}}、\ce{Ca^{2+}}、\ce{Ag+}
\ce{CO3^{2-}}、\ce{SO3^{2-}}、\ce{PO4^{3-}}、\ce{S^{2-}} 除 \ce{Na+}、\ce{K+}、\ce{NH4+} 外的大多数阳离子
\ce{OH-} 除 \ce{Na+}、\ce{K+}、\ce{Ba^{2+}} 外的大多数阳离子

注：高中阶段常见的四种微溶物包括 \ce{Ca(OH)2}、\ce{CaSO4}、\ce{Ag2SO4}、\ce{MgCO3}。

常见离子的检验

离子 检验方法 易错点
\ce{H+} 使紫色石蕊溶液变红；用 \ce{pH} 试纸测定；与 \ce{Zn}、\ce{Fe} 等反应放出氢气 \ce{pH} 试纸的使用方法：放、蘸、点、显、读
\ce{OH-} 使紫色石蕊溶液变蓝，酚酞溶液变红；用 \ce{pH} 试纸测定； \ce{pH} 试纸的使用方法：放、蘸、点、显、读
