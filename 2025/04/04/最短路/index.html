无权图最短路（BFS）

广度优先搜索（BFS）。

时间复杂度：O(n)O(n)O(n)。

无负环单源最短路径（Dijkstra算法）

Dijkstra算法本质是贪心，从一个点出发到所有点的最短路径均需要求出。

适用范围：边权必须是非负整数。

时间复杂度：O(n2+m)O(n^2+m)O(n2+m)。（适用于稠密图）

做法

  1. 初始化：dis[i] 代表目前为止，从起点到 iii 号节点的最短距离。dis[i]=INF, dis[s]=0;。
  2. 从 111 到 n−1n-1n−1：（1）找未确定中 dis[i] 的最小值。（2）将最小值标记为已确定。（3）松弛操作。
  3. 松弛操作：if (u.dis+w<v.dis) v.dis=u.dis+w;。

题目描述

输入格式

接下来 mmm 行每行包含三个整数 u,v,wu,v,wu,v,w，表示一条 u→vu \to vu→v 的，长度为 www 的边。

输出一行 nnn 个整数，第 iii 个表示 sss 到第 iii 个点的最短路径，若不能到达则输出 231−12^{31}-1231−1。

有负环单源最短路径（SPFA算法）

时间复杂度：O(km)O(km)O(km)（最坏时间复杂度为 O(nm)O(nm)O(nm)）。

适用范围：图中有负边权要判断负环。

如果确定图中边长非负且没有特殊限制时，不应使用SPFA算法。

做法

  1. 将起始点的 disdisdis 设为 000 并放入队列。
  2. 选择当前队列中的第一个点并将其从队列中删除。
  3. 对该点的所有连边依次进行松弛操作。
  4. 将第三步中松弛成功且不在队列内的点放入队列。
  5. 重复第二步至第四步，直到队列为空。

多源最短路经路径（Floyd算法）

任意两点间的最短路都需要求出。

适用范围：多源最短路。

时间复杂度：O(n3)O(n^3)O(n3)。空间复杂度：O(n3)O(n^3)O(n3)。

做法

采用动态规划的思路。

状态设定：设 disi,j,kdis_{i,j,k}disi,j,k​ 表示当只途径编号为 111 到 kkk 的点时（起始点和终点不算），iii 到 jjj 的最短路长度。

初始值：disi,j,0dis_{i,j,0}disi,j,0​ 表示初始时 iii、jjj 点间的边的边权。

状态转移：disi,j,k=min⁡(disi,j,k−1,disi,k,k−1+disk,j,k−1)dis_{i,j,k}=\min(dis_{i,j,k-1},dis_{i,k,k-1}+dis_{k,j,k-1})disi,j,k​=min(disi,j,k−1​,disi,k,k−1​+disk,j,k−1​)。

优化

显然 disi,k,k−1dis_{i,k,k-1}disi,k,k−1​ 与 disi,k,kdis_{i,k,k}disi,k,k​、disk,j,k−1dis_{k,j,k-1}disk,j,k−1​ 与 disk,j,kdis_{k,j,k}disk,j,k​ 分别相等，于是发现数组第三维 kkk 时多余的，可以直接用二维数组来解决。

状态转移方程变为：disi,j=min(disi,j,disi,k+disk,j)dis_{i,j}=min(dis_{i,j},dis_{i,k}+dis_{k,j})disi,j​=min(disi,j​,disi,k​+disk,j​)。

空间复杂度：O(n2)O(n^2)O(n2)。

题目描述

给出一张由 nnn 个点 mmm 条边组成的无向图。

求出所有点对 (i,j)(i,j)(i,j) 之间的最短路径。

输入格式

第一行为两个整数 n,mn,mn,m，分别代表点的个数和边的条数。

接下来 mmm 行，每行三个整数 u,v,wu,v,wu,v,w，代表 u,vu,vu,v 之间存在一条边权为 www 的边。

输出格式

输出 nnn 行每行 nnn 个整数。

第 iii 行的第 jjj 个整数代表从 iii 到 jjj 的最短路径。

数据范围

对于 100%100\%100% 的数据，n≤100n \le 100n≤100，m≤4500m \le 4500m≤4500，任意一条边的权值 www 是正整数且 1⩽w⩽10001 \leqslant w \leqslant 10001⩽w⩽1000。
