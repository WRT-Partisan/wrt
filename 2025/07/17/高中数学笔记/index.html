初高衔接

等式变形

乘法公式

  1. a2−b2=(a+b)(a−b)a^2-b^2=(a+b)(a-b)a2−b2=(a+b)(a−b)
  2. (a±b)2=a2±2ab+b2(a\pm b)^2=a^2\pm 2ab+b^2(a±b)2=a2±2ab+b2
  3. (a±b)3=a3±3a2b+3ab2±b3(a\pm b)^3=a^3\pm 3a^2b+3ab^2\pm b^3(a±b)3=a3±3a2b+3ab2±b3
  4. a3+b3=(a+b)(a2−ab+b2)a^3+b^3=(a+b)(a^2-ab+b^2)a3+b3=(a+b)(a2−ab+b2)
  5. a3−b3=(a−b)(a2+ab+b2)a^3-b^3=(a-b)(a^2+ab+b^2)a3−b3=(a−b)(a2+ab+b2)

在化简根式时，如果遇到根号套根号的情况，考虑将最外层根号下的式子化成完全平方式再进行化简，注意化简后要带绝对值符号。

因式分解

公式法

  1. a2−b2=(a+b)(a−b)a^2-b^2=(a+b)(a-b)a2−b2=(a+b)(a−b)
  2. a2±2ab+b2=(a±b)2a^2\pm 2ab+b^2=(a\pm b)^2a2±2ab+b2=(a±b)2
  3. a3±3a2b+3ab2±b3=(a±b)3a^3\pm 3a^2b+3ab^2\pm b^3=(a\pm b)^3a3±3a2b+3ab2±b3=(a±b)3
  4. a3+b3=(a+b)(a2−ab+b2)a^3+b^3=(a+b)(a^2-ab+b^2)a3+b3=(a+b)(a2−ab+b2)
  5. a3−b3=(a−b)(a2+ab+b2)a^3-b^3=(a-b)(a^2+ab+b^2)a3−b3=(a−b)(a2+ab+b2)

十字相乘法

常用在二次三项式、二次齐次式、二元二次式中。

Ax2+Bx+C=(ax+b)(cx+d)=acx2+(ad+bc)x+bdAx^2+Bx+C=(ax+b)(cx+d)=acx^2+(ad+bc)x+bdAx2+Bx+C=(ax+b)(cx+d)=acx2+(ad+bc)x+bd

⇒ac=A,bd=C,ad+bc=B\Rightarrow ac=A,bd=C,ad+bc=B⇒ac=A,bd=C,ad+bc=B

方法：

  1. 分解二次项系数为两数/式相乘，写在左边。
  2. 分解常数项为两数/式相乘，写在右边。
  3. 若交叉相乘再相加的结果等于一次项系数，则该分解方式成立；若不相等，则改变分解方式或调换上下位置，多次尝试，直到找出分解方法。

二次齐次式同理：Ax2+Bxy+Cy2=(ax+by)(cx+dy)=acx2+(ad+bc)xy+bdy2Ax^2+Bxy+Cy^2=(ax+by)(cx+dy)=acx^2+(ad+bc)xy+bdy^2Ax2+Bxy+Cy2=(ax+by)(cx+dy)=acx2+(ad+bc)xy+bdy2

若遇到二元二次式，则可以用主元法，即设定其中一元为主元，写成关于主元二次三项式，并用分解二次三项式的方法分解。

增添项法

常用在次数较高的多项式中，此时可以增添项并分组分解。

  1. 猜根：如 x=±1,±2,±3,±12,±13x=\pm 1,\pm 2,\pm 3,\pm\dfrac{1}{2},\pm\dfrac{1}{3}x=±1,±2,±3,±21​,±31​。
  2. 将原式按照猜到的因式进行增添项，由高次到低次。
  3. 分组分解，提出之前猜到的因式，合并，整理，注意检查是否分解彻底。

不等式解法

一元二次不等式

  1. 解出相应的一元二次方程。
  2. 在平面直角坐标系中画出相应二次函数的图象。
  3. 根据图象找出不等式的解集。

分式不等式

一般不等式：ab>0⇔a与b同号⇔ab>0\dfrac{a}{b}>0\Leftrightarrow a\text{与}b\text{同号}\Leftrightarrow ab>0ba​>0⇔a与b同号⇔ab>0，将其转化为整式不等式再求解，小于 000 同理。

含等号的不等式：ab≥0⇔ab≥0(b≠0)\dfrac{a}{b}\ge 0\Leftrightarrow ab\ge 0(b\ne 0)ba​≥0⇔ab≥0(b=0)，将其转化为整式不等式再求解，注意分母不能为 000，小于等于 000 同理。

右侧不为 000 的不等式：ab>c⇔ab−c>0⇔a−bcb>0\dfrac{a}{b}>c\Leftrightarrow \dfrac{a}{b}-c>0\Leftrightarrow \dfrac{a-bc}{b}>0ba​>c⇔ba​−c>0⇔ba−bc​>0，移项后同上；或者两边同乘分母，根据符号分类讨论。

高次不等式

化为一般形式后因式分解，算出其根，根即为与 xxx 轴的交点。

穿过根画出图象，需要带特值确定图象的方向，寻找解集。

如果该根所在的项为奇数次，那么图象穿过根；如果该根所在的项为偶数次，那么图象不穿过根。

即：

  1. 先算根。
  2. 判断穿根方向（代值）。
  3. 奇穿偶不穿。

绝对值不等式

只有一个绝对值且另一侧是常数

根据绝对值几何定义，再数轴上表示出绝对值内式子的范围，再分别求解。

两边都有绝对值

∣a∣>∣b∣⇔a2>b2\left|a\right|>\left|b\right|\Leftrightarrow a^2>b^2∣a∣>∣b∣⇔a2>b2，转化为一元二次不等式，移项构造平方差公式。

只有一个绝对值且另一侧含未知数

去绝对值，分大于 000 和小于 000 两种情况讨论。

根式不等式

两边同时平方，需要注意符号。

a<b⇔a<b2(a≥0,b>0)\sqrt{a}<b\Leftrightarrow a<b^2(a\ge 0,b>0)a​<b⇔a<b2(a≥0,b>0)。

必修一

集合

集合的概念

集合：研究对象所汇总成的集体，通常用大写字母表示。

元素：构成集合的每个研究对象，通常用小写字母表示。

两集合相等：构成两个集合的元素完全一样。

如果 aaa 是集合 AAA 的元素，就说 aaa 属于 AAA，记作 a∈Aa\in Aa∈A；如果 aaa 不是集合 AAA 的元素，就说 aaa 不属于 AAA，记作 a∉Aa\notin Aa∈/A。

集合的三大特性

  1. 确定性：集合的元素必须是确定的，即有统一标准判断一个元素是否属于这个集合。
  2. 无序性：集合中的元素可以任意排列。
  3. 互异性：对于给定的一个集合，集合中的元素必须是不同的（注意集合内部有参数）。

常见数集

  1. 自然数集：N\mathbb{N}N。
  2. 正整数集：N+\mathbb{N}^+N+ 或 N∗\mathbb{N}^*N∗。
  3. 整数集：Z\mathbb{Z}Z。
  4. 有理数集：Q\mathbb{Q}Q。
  5. 实数集：R\mathbb{R}R。

集合的表示方法

列举法

把集合的所有元素一一列举出来，用逗号“,,,”隔开，并用大括号“{}\{\}{}”括起来。若无法列举完且规律明显可用省略号“…\dots…”表示。

描述法

在大括号中画一条竖线，在竖线前面写研究对象，在竖线后面写特征，即 {研究对象∣特征}\{\text{研究对象}|\text{特征}\}{研究对象∣特征}，若有多个特征中间用逗号“,,,”隔开。

在数学中一般研究对象为数，即 xxx，一般表示方法为 {x∈数集∣特征}\{x\in\text{数集}|\text{特征}\}{x∈数集∣特征}，为数集。

点集用坐标表示，一般表示方法为 {(x,y)∣点的特征}\{(x,y)|\text{点的特征}\}{(x,y)∣点的特征}。

集合间关系

空集的概念

空集：不含任何元素的集合，表示为“∅\varnothing∅”。

空集经常作为题目中的陷阱出现，集合里有参数，需要注意空集。

若出现取值范围则需要考虑等号是否能取到，可以代值试，答案需写成集合的形式。

子集的概念

子集：如果集合 AAA 中任意一个元素都属于集合 BBB，则称集合 AAA 是集合 BBB 的子集，记作 A⊆BA\subseteq BA⊆B 或 B⊇AB\supseteq AB⊇A。

A=BA=BA=B 是 A⊆BA\subseteq BA⊆B 的特殊情况，此时 B⊆AB\subseteq AB⊆A 也成立。

真子集：如果 A⊆BA\subseteq BA⊆B，但存在元素 x∈Bx\in Bx∈B，且 x∉Ax\notin Ax∈/A，则称集合 AAA 是集合 BBB 的真子集，记作 A⫋BA\subsetneqq BA⫋B 或 B⫌AB\supsetneqq AB⫌A。

规定：∅\varnothing∅ 是任意集合的子集，∅\varnothing∅ 是任意非空集合的真子集，∅\varnothing∅ 是 ∅\varnothing∅ 的子集，∅\varnothing∅ 不是 ∅\varnothing∅ 的真子集。

遇到子集或真子集需要考虑空集的情况。

AAA 是 BBB 的子集有三种情况：

  1. A⫋BA\subsetneqq BA⫋B。
  2. A=BA=BA=B。
  3. A=∅A=\varnothingA=∅。
子集的个数

对于有 nnn 个元素的非空集合 AAA，其中 n≥1n\ge 1n≥1：

  1. 子集个数：2n2^n2n。
  2. 真子集个数：2n−12^n-12n−1。
  3. 非空子集个数：2n−12^n-12n−1。
  4. 非空真子集个数：2n−22^n-22n−2。
已知子集关系求参数范围

在数轴上进行分析，列出不等式，注意判断是否应该加等号。

需要特别注意空集的情况，参数范围需写成集合的形式。

集合的抽象表达式

Venn 图：用平面上封闭曲线的内部代表集合。

遇到集合的抽象表达式且没有固定元素的集合关系可通过画 Venn 图分析。

集合的运算

交集

交集：表示两个集合的重叠部分（与）。

AAA 与 BBB 的交集为所有属于集合 AAA 且属于集合 BBB 的元素组成的集合，记作 A∩BA\cap BA∩B，读作“AAA 交 BBB”。

并集

并集：表示两个集合的所有部分（或）。

AAA 与 BBB 的并集为所有属于集合 AAA 或属于集合 BBB 的元素组成的集合，记作 A∪BA\cup BA∪B，读作“AAA 并 BBB”。

补集

全集：表示所有范围，即含有所研究问题中涉及的所有元素的集合。

补集：表示全集中集合外的部分（非）。

AAA 相对于全集 UUU 的补集为由全集 UUU 中不属于集合 AAA 的所有元素组成的集合，简称为 AAA 的补集，记作 ∁UA\complement_UA∁U​A。

∁UA={x∣x∈U且x∉A}\complement_UA=\{x|x\in U\text{且}x\notin A\}∁U​A={x∣x∈U且x∈/A}。

集合综合

取交集和并集的时机：一类之内取交集，类类之间取并集。
