十字相乘法

常用在二次三项式,二次齐次式,二元二次式中.

Ax2+Bx+C=(ax+b)(cx+d)=acx2+(ad+bc)x+bdAx^2+Bx+C=(ax+b)(cx+d)=acx^2+(ad+bc)x+bdAx2+Bx+C=(ax+b)(cx+d)=acx2+(ad+bc)x+bd

⇒ac=A,bd=C,ad+bc=B\Rightarrow ac=A,bd=C,ad+bc=B⇒ac=A,bd=C,ad+bc=B

方法:

  1. 分解二次项系数为两数/式相乘,写在左边.
  2. 分解常数项为两数/式相乘,写在右边.
  3. 若交叉相乘再相加的结果等于一次项系数,则该分解方式成立;若不相等,则改变分解方式或调换上下位置,多次尝试,直到找出分解方法.

二次齐次式同理:Ax2+Bxy+Cy2=(ax+by)(cx+dy)=acx2+(ad+bc)xy+bdy2Ax^2+Bxy+Cy^2=(ax+by)(cx+dy)=acx^2+(ad+bc)xy+bdy^2Ax2+Bxy+Cy2=(ax+by)(cx+dy)=acx2+(ad+bc)xy+bdy2

若遇到二元二次式,则可以用主元法,即设定其中一元为主元,写成关于主元二次三项式,并用分解二次三项式的方法分解.

增添项法

常用在次数较高的多项式中,此时可以增添项并分组分解.

  1. 猜根:如 x=±1,±2,±3,±12,±13x=\pm 1,\pm 2,\pm 3,\pm\frac{1}{2},\pm\frac{1}{3}x=±1,±2,±3,±21​,±31​.
  2. 将原式按照猜到的因式进行增添项,由高次到低次.
  3. 分组分解,提出之前猜到的因式,合并,整理,注意检查是否分解彻底.

集合

集合的概念

集合:研究对象所汇总成的集体,通常用大写字母表示.

元素:构成集合的每个研究对象,通常用小写字母表示.

两集合相等:构成两个集合的元素完全一样.

如果 aaa 是集合 AAA 的元素,就说 aaa 属于 AAA,记作 a∈Aa\in Aa∈A;如果 aaa 不是集合 AAA 的元素,就说 aaa 不属于 AAA,记作 a∉Aa\notin Aa∈/A.

集合的三大特性

  1. 确定性:集合的元素必须是确定的,即有统一标准判断一个元素是否属于这个集合.
  2. 无序性:集合中的元素可以任意排列.
  3. 互异性:对于给定的一个集合,集合中的元素必须是不同的(注意集合内部有参数).

常见数集

  1. 自然数集:N\mathbb{N}N.
  2. 正整数集:N+\mathbb{N}^+N+ 或 N∗\mathbb{N}^*N∗.
  3. 整数集:Z\mathbb{Z}Z.
  4. 有理数集:Q\mathbb{Q}Q.
  5. 实数集:R\mathbb{R}R.

集合的表示方法

列举法

把集合的所有元素一一列举出来,用逗号",,,“隔开,并用大括号”{}\{\}{}“括起来.若无法列举完且规律明显可用省略号”…\dots…"表示.

描述法

在大括号中画一条竖线,在竖线前面写研究对象,在竖线后面写特征,即 {研究对象∣特征}\{\text{研究对象}|\text{特征}\}{研究对象∣特征},若有多个特征中间用逗号",,,"隔开.

在数学中一般研究对象为数,即 xxx,一般表示方法为 {x∈数集∣特征}\{x\in\text{数集}|\text{特征}\}{x∈数集∣特征},为数集.

点集用坐标表示,一般表示方法为 {(x,y)∣点的特征}\{(x,y)|\text{点的特征}\}{(x,y)∣点的特征}.

集合间关系

空集的概念

空集:不含任何元素的集合,表示为"∅\varnothing∅".

空集经常作为题目中的陷阱出现,集合里有参数,需要注意空集.

若出现取值范围则需要考虑等号是否能取到,可以代值试,答案需写成集合的形式.

子集的概念

子集:如果集合 AAA 中任意一个元素都属于集合 BBB,则称集合 AAA 是集合 BBB 的子集,记作 A⊆BA\subseteq BA⊆B 或 B⊇AB\supseteq AB⊇A,读作"AAA 包含于 BBB"或"BBB 包含 AAA".

A=BA=BA=B 是 A⊆BA\subseteq BA⊆B 的特殊情况,此时 B⊆AB\subseteq AB⊆A 也成立.

真子集:如果 A⊆BA\subseteq BA⊆B,但存在元素 x∈Bx\in Bx∈B,且 x∉Ax\notin Ax∈/A,则称集合 AAA 是集合 BBB 的真子集,记作 A⫋BA\subsetneqq BA⫋B 或 B⫌AB\supsetneqq AB⫌A,读作"AAA 真包含于 BBB"或"BBB 真包含 AAA".

规定:∅\varnothing∅ 是任意集合的子集,∅\varnothing∅ 是任意非空集合的真子集,∅\varnothing∅ 是 ∅\varnothing∅ 的子集,∅\varnothing∅ 不是 ∅\varnothing∅ 的真子集.

遇到子集或真子集需要考虑空集的情况.

AAA 是 BBB 的子集有三种情况:

  1. A⫋BA\subsetneqq BA⫋B.
  2. A=BA=BA=B.
  3. A=∅A=\varnothingA=∅.

子集的个数

对于有 nnn 个元素的非空集合 AAA,其中 n≥1n\ge 1n≥1:

  1. 子集个数:2n2^n2n.
  2. 真子集个数:2n−12^n-12n−1.
  3. 非空子集个数:2n−12^n-12n−1.
  4. 非空真子集个数:2n−22^n-22n−2.

已知子集关系求参数范围

在数轴上进行分析,列出不等式,注意判断是否应该加等号.

需要特别注意空集的情况,参数范围需写成集合的形式.

集合的运算

交集

交集:表示两个集合的重叠部分(与).

AAA 与 BBB 的交集为所有属于集合 AAA 且属于集合 BBB 的元素组成的集合,记作 A∩BA\cap BA∩B,读作"AAA 交 BBB".

并集

并集:表示两个集合的所有部分(或).

AAA 与 BBB 的并集为所有属于集合 AAA 或属于集合 BBB 的元素组成的集合,记作 A∪BA\cup BA∪B,读作"AAA 并 BBB".

补集

全集:表示所有范围,即含有所研究问题中涉及的所有元素的集合.

补集:表示全集中集合外的部分(非).

AAA 相对于全集 UUU 的补集为由全集 UUU 中不属于集合 AAA 的所有元素组成的集合,简称为 AAA 的补集,记作 ∁UA\complement_UA∁U​A.

∁UA={x∣x∈U且x∉A}\complement_UA=\{x|x\in U\text{且}x\notin A\}∁U​A={x∣x∈U且x∈/A}.

集合综合

取交集和并集的时机:一类之内取交集,类类之间取并集.

集合易错点

符号,概念混淆不清

属于和包含于

属于:∈\in∈,是元素与集合之间的关系.

包含于:⊆\subseteq⊆,是集合与集合之间的关系.

0 和空集

000:元素 000.

{0}\{0\}{0}:只有元素 000 的集合.

∅\varnothing∅:空集.

{∅}\{\varnothing\}{∅}:集合套集合(222 层),即只有元素空集的集合.

表示方程组的解

不能表示成 {x=a,y=b}\{x=a,y=b\}{x=a,y=b} 的形式,应该表示成 {(a,b)}\{(a,b)\}{(a,b)} 或 {(x,y)∣x=a,y=b}\{(x,y)|x=a,y=b\}{(x,y)∣x=a,y=b} 的形式.

复杂描述法

看清楚研究对象是什么,分清楚数集和点集,若遇到点集往往需要画函数图象.

列举法陷阱

列举法里有参数,应注意集合的互异性,可能需要分类讨论.

分类讨论

最高次项有参数,应考虑是否为 000.

如遇到题目有子集关系应考虑空集.

常用逻辑用语

充分必要条件

由条件推结论的过程叫做充分性推导,由结论推条件的过程叫做必要性推导.

如果 ppp 能推出 qqq,那么称 ppp 是 qqq 的充分条件,qqq 是 ppp 的必要条件,记作 p⇒qp\Rightarrow qp⇒q.

如果 ppp 不能推出 qqq,那么称 ppp 不是 qqq 的充分条件,qqq 不是 ppp 的必要条件,记作 p⇏qp\nRightarrow qp⇏q.

若 ppp 能推出 qQqQqQ,qqq 也能推出 ppp,那么称 ppp 和 qqq 互为充分必要条件,简称充要条件,记作 p⇔qp\Leftrightarrow qp⇔q.

小范围可以推大范围,大范围不能推消小范围.

易错点:注意看清谁是条件

ppp 是 qqq 的必要不充分条件:Q⫋PQ\subsetneqq PQ⫋P.

ppp 的必要不充分条件是 qqq:P⫋QP\subsetneqq QP⫋Q.

全称与存在量词

全称量词与存在量词的概念

全称量词:表示"所有",记作 ∀\forall∀.

存在量词:表示"存在",记作 ∃\exists∃.

全称量词命题与存在量词命题的否定

否定:用符号 ¬p(x)\neg p(x)¬p(x) 表示 p(x)p(x)p(x) 不成立.

全称量词命题 ∀x∈M,p(x)\forall x\in M,p(x)∀x∈M,p(x),它的否定形式为 ∃x∈M,¬p(x)\exists x\in M,\neg p(x)∃x∈M,¬p(x).

存在量词命题 ∃x∈M,p(x)\exists x\in M,p(x)∃x∈M,p(x),它的否定形式为 ∀x∈M,¬p(x)\forall x\in M,\neg p(x)∀x∈M,¬p(x).

一个命题的否定形式应将全称量词和存在量词交换,条件不变,结论取反,结论中出现"或"需要变成"且",结论中出现"且"需要变成或.

题目中若遇到"假命题"等价于该命题的反命题是真命题.

不等式

不等式性质

  1. 加减:a>b⇒a±c>b±ca>b\Rightarrow a\pm c>b\pm ca>b⇒a±c>b±c.
  2. 乘除正数:a>b,c>0⇒ac>bc,ac>bca>b,c>0\Rightarrow ac>bc,\frac{a}{c}>\frac{b}{c}a>b,c>0⇒ac>bc,ca​>cb​.
  3. 乘除负数:a>b,c<0⇒ac<bc,ac<bca>b,c<0\Rightarrow ac<bc,\frac{a}{c}<\frac{b}{c}a>b,c<0⇒ac<bc,ca​<cb​.
  4. 同向可加:a>b,c>d⇒a+c>b+da>b,c>d\Rightarrow a+c>b+da>b,c>d⇒a+c>b+d.
  5. 同向同正可乘:a>b>0,c>d>0⇒ac>bda>b>0,c>d>0\Rightarrow ac>bda>b>0,c>d>0⇒ac>bd.
  6. 正数乘方开方:a>b>0,,n≥1⇒an>bn,an>bna>b>0,,n\ge 1\Rightarrow a^n>b^n,\sqrt[n]{a}>\sqrt[n]{b}a>b>0,,n≥1⇒an>bn,na​>nb​.

选择题做题方法:

  1. 根据不等式的性质直接判断.
  2. 将不等式化简.
  3. 代入特殊值判断.

解答题做题方法:

比较两个式子的大小通常需要将两个式子作差.若式子无法作差(如带根号)应先变形再作差.

  1. a−b>0⇒a>ba-b>0\Rightarrow a>ba−b>0⇒a>b.
  2. a−b=0⇒a=ba-b=0\Rightarrow a=ba−b=0⇒a=b.
  3. a−b<0⇒a<ba-b<0\Rightarrow a<ba−b<0⇒a<b.

由不等式求范围问题:

若遇到基本元素为中项的不等式直接凑成要求的式子.

若遇到代数式为中项则将代数式换元(把一个复杂的式子换成简单的变量的方法叫做"换元"),联立方程解出各个元素的值,再将元素代换成元素的值凑出要求的式子.

基本不等式

一般形式

基本不等式的推导过程:

(x−y)2≥0⇒x+y−2xy≥0⇒x+y≥2xy(x,y≥0)(\sqrt{x}-\sqrt{y})^2\ge 0\Rightarrow x+y-2\sqrt{xy}\ge 0\Rightarrow x+y\ge 2\sqrt{xy}(x,y\ge 0)(x​−y​)2≥0⇒x+y−2xy​≥0⇒x+y≥2xy​(x,y≥0).

基本不等式的一般形式:

x+y≥2xy(x,y≥0)x+y\ge 2\sqrt{xy}(x,y\ge 0)x+y≥2xy​(x,y≥0).

基本不等式实质上是两个数/式的和与两个数/式的积之间的不等关系,在题目中一般用于求最值.

不等式的取等问题

x+y≥2xy(x,y≥0)x+y\ge 2\sqrt{xy}(x,y\ge 0)x+y≥2xy​(x,y≥0),当且仅当 x=yx=yx=y 时,等号成立.

注意求最值时要验证等号是否能成立.

和积式

若在题目中既有两式的和又有两式的积,则可以直接运用基本不等式,即两式的和大于等于二倍根号两式的积,注意两式都为非负,并注意对取等的判断.

有积定求和与和定求积两种情况.

有和有积求最值,想基本不等式.

非齐次式

多项式中每项的次数不一样,可以通过基本不等式抵消未知数直接求出最值.

若底数不一致则可以尝试凑出相同的底数,再用基本不等式.

齐次式

齐次式:分子分母每项次数都相同的多项式,可以通过基本不等式抵消未知数直接求出最值.

若分子比较复杂分母比较简单可考虑拆分子的方法.

若分母比较复杂可以考虑换元法(整体代换).

"111"的代换:若题目中给定条件代数式等于常数,则可以考虑将所求的代数式中的常数代换成代数式,或者将所求的代数式乘上给定条件代数式再除以常数.

注意:1a+1b=1⇔a+b=ab⇔(a−1)(b−1)=1\frac{1}{a}+\frac{1}{b}=1\Leftrightarrow a+b=ab\Leftrightarrow (a-1)(b-1)=1a1​+b1​=1⇔a+b=ab⇔(a−1)(b−1)=1.

若等式一边为一次幂,另一边为二次幂,则两边同除以二次幂获得负一次幂等于一,将一次幂乘一(即负一次幂)得到齐次式,然后用基本不等式.

换元法

在和积式中遇到积复杂(如带根号)则可以将复杂的因式换元.

若分母比较复杂(如不为单项式)则可以将复杂的分母换元.

凑齐次

若分子的最高次数与分母的最高次数相等,或可以简单地通过"111"的代换来升次,则可以考虑凑齐次式,将其他低于最高次数的项通过"111"的代换进行升次,转化为齐次式.

强行消元(下策)

所有方法都是投机取巧,只有硬算才是真理!

若题目太难或实在没有思路,则可以将给出的表达式转化为一个未知数等于含另一个未知数的式子的形式,将所求代数式中的这个未知数代换为转化出的代数式,进行强行消元后再计算.

多次利用基本不等式

既然已经用了,那么再用几次又有何妨?

用基本不等式求式子最值的问题根源上是解方程的问题,需要把每个未知数的取值确定出来才能确定式子的最值,而每用一次基本不等式都会产生一个取等条件(即一个方程),式子中元的个数与方程的个数是相等的,由此可以确定使用基本不等式的次数.

若式子中的未知数较多,可以将具有未知数相同或大致相同的项放在一起,并进行配凑乱搞(超强注意力),尝试凑分母或一些奇妙的东西(如完全平方式式);也可以运用换元法,使分母变简单.

注意尤其是在用多次基本不等式后需要验证是否存在解使所有等号成立.

二次函数图象及解析式

确定二次函数图象需要注意四个方面:aaa,对称轴(−b2a-\frac{b}{2a}−2ab​),Δ\DeltaΔ(b2−4acb^2-4acb2−4ac),ccc.

二次函数一共有三种解析式:

  1. 一般式:y=ax2+bx+c(a≠0)y=ax^2+bx+c(a\ne 0)y=ax2+bx+c(a=0),其中 ccc 为图象与纵轴交点的纵坐标.
  2. 顶点式:y=a(x−h)2+k(a≠0)y=a(x-h)^2+k(a\ne 0)y=a(x−h)2+k(a=0),其中 (h,k)(h,k)(h,k) 为图象最高/最低点的坐标.
  3. 交点式:y=a(x−x1)(x−x2)(a≠0)y=a(x-x_1)(x-x_2)(a\ne 0)y=a(x−x1​)(x−x2​)(a=0),其中 x1x_1x1​ 和 x2x_2x2​ 是图象与横轴两个交点的横坐标.

韦达定理/一元二次方程根与系数关系:如果一元二次方程 Ax2+Bx+C=0(A≠0)Ax^2+Bx+C=0(A\ne 0)Ax2+Bx+C=0(A=0) 有两个解 x1x_1x1​,x2x_2x2​,则 x1+x2=−BAx_1+x_2=-\frac{B}{A}x1​+x2​=−AB​,x1x2=CAx_1x_2=\frac{C}{A}x1​x2​=AC​.

一元二次不等式的解法

  1. 将一元二次不等式转化为一般形式,将不等式左边因式分解(如果易因式分解)或求出其两个根.
  2. 将不等式左边的函数图象在数轴(横轴)上大致画出来,并标明图象与横轴的两个交点,注意 aaa 的正负决定图象的开口方向.
  3. 根据不等号和函数图象判断不等式的解集,注意是否取等.

高次不等式

化为一般形式后因式分解,算出其根,根即为与 xxx 轴的交点.

  2. 判断穿根方向(代值).
  3. 奇穿偶不穿.
