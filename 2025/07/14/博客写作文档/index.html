写作规范

请在发布文章前检查文章是否满足以下规范。

  1. 标题尽量不要使用一级标题，从二级标题开始。
  2. 单词/公式和中文之间要加空格。
  3. 中文要使用全角标点，末尾要句号；单词/公式和全角标点中无空格。
  4. 所有数学符号、常数、作为变量的字母应使用 LaTeX\LaTeXLATE​X 公式。
  5. 行内字符串、代码片段用 ` 包裹。
  6. 不要把 LaTeX\LaTeXLATE​X 公式滥用在非公式的单词或者缩写上。
  7. 要创建段落，请使用空白行将一行或多行文本进行分隔，不要用空格（spaces）或制表符（tabs）缩进段落，如有特殊情况可使用 &nbsp; 缩进。
  8. 除特殊情况尽量使用严格换行，段落与段落间、段落与标题间应有两个换行符，段落内不应有换行符。
  9. 粗体和斜体尽量使用星号，不要使用下划线。
  10. 块引用可以包含多个段落，可在段落之间的空白行添加一个 > 符号。段落内不应出现换行和 > 符号。
  11. 有序列表数字尽量有序，无序列表尽量使用 - 符号。
  12. 代码块应标明代码语言。
  13. 为了兼容性，请在分隔线的前后均添加空白行。
  14. 链接和图片应尽量写上超链接显示名或图片名。

写作技巧

  1. 块引用可以嵌套，在要嵌套的段落前添加一个 >> 符号。
  2. 块引用可以包含其他部分 Markdown 格式的元素。
  3. 列表中可以嵌入其他元素，要在保留列表连续性的同时在列表中添加另一种元素，请将该元素缩进四个空格或一个制表符。代码块通常采用四个空格或一个制表符缩进。当它们被放在列表中时，请将它们缩进八个空格或两个制表符。
  4. 如果你要表示为代码的单词或短语中包含一个或多个反引号，则可以通过将单词或短语包裹在双反引号(``)中。
  5. 链接 title 是当鼠标悬停在链接上时会出现的文字，这个 title 是可选的，它放在圆括号中链接地址后面，跟链接地址之间以空格分隔，并用双引号包裹。
  6. 使用尖括号可以很方便地把 URL 或者 email 地址变成可点击的链接。
  7. 强调链接, 在链接语法前后增加星号。要将链接表示为代码，请在方括号中添加反引号。
  8. 给图片增加链接，请将图像的 Markdown 括在方括号中，然后将链接添加在圆括号中。
  9. 对于 Markdown 涵盖范围之外的标签，都可以直接在文件里面用 HTML 本身。如需使用 HTML，不需要额外标注这是 HTML 或是 Markdown，只需 HTML 标签添加到 Markdown 文本中即可。
  10. 区块元素──比如 <div>、<table>、<pre>、<p> 等标签，必须在前后加上空行，以便于内容区分。而且这些元素的开始与结尾标签，不可以用 tab 或是空白来缩进。
  11. 可以通过在标题行中的连字符的左侧，右侧或两侧添加冒号（:），将列中的文本对齐到左侧，右侧或中心。可以在表格中设置文本格式，如链接，代码（仅反引号（`）中的单词或短语，而不是代码块）和强调。可以使用表格的HTML字符代码（|）在表中显示竖线（|）字符。
  12. 链接到标题 ID：[Heading IDs](#heading-ids)。
  13. 创建定义列表，请在第一行上键入术语。在下一行，键入一个冒号，后跟一个空格和定义。
  14. Markdown 不能直接嵌入视频，但可以使用 HTML。点击 B 站的分享图标，然后选择嵌入链接，会获得 HTML 代码。

LaTeX 使用规范

LaTeX\LaTeXLATE​X 的公式是非常繁杂的，其中所涉及的宏包数不胜数，如有需要应查看相关文档或搜寻相关资料。

  1. 同一个数学公式应写在一个 LaTeX\LaTeXLATE​X 环境内。
  2. 所有数学符号、常数、作为变量的字母应使用 LaTeX\LaTeXLATE​X 公式，用 $ 包裹。
  3. 公式汉字中间加空格，但和标点中间不用。
  4. 大数字应使用科学计数法。
  5. 常见符号如乘号应使用 LaTeX\LaTeXLATE​X 命令，不能直接输入或复制。
  6. 常见函数在 LaTeX\LaTeXLATE​X 中都有专门的表示，不应直接输入。
  7. 数学公式中的文本应使用 \text，字符串应使用 \texttt。
  8. 公式独立成行时应使用行间公式，即 $$formula$$。
  9. 数学公式中应使用数学语言而非代码语言。
  10. 特定的、约定俗成的函数名称应该使用正体。对于一些未定义的函数，应使用 \operatorname。
  11. 应正确使用大型运算符，建议在可能引起误解的地方加上括号。
  12. 波浪线应使用 $\sim$。
  13. 连等式应使用 \aligned 环境，分段函数应使用 \cases 环境，矩阵应使用 \bmatrix 环境。

Mermaid

使用 mermaid 标签可以绘制 Flowchart（流程图）、Sequence diagram（时序图 ）、Class Diagram（类别图）、State Diagram（状态图）、Gantt（甘特图）和 Pie Chart（圆形图），具体可以查看 mermaid 文档。

Tabs

参数 解释
Unique name tabs 区块标签的唯一名称，不包含逗号。将用于每个选项卡的 #id 前缀，并附加其索引号。若名称中有空格，生成 #id 时会将空格替换为短横线。仅对当前文章/页面的 URL 必须唯一！
[index] 【可选】活动选项卡的索引号。如果未指定，将选择第一个选项卡（1）。如果索引为 -1，则不会选择任何选项卡，类似于折叠内容。可选参数。
[Tab caption] 当前选项卡的标题。如果未指定标题，将使用唯一名称和选项卡索引后缀作为选项卡标题。如果未指定标题，但指定了图标，标题将为空。
[@icon] 【可选】FontAwesome 图标名称（全名，如 ‘fas fa-font’）。可以有或没有空格；例如 ‘Tab caption @icon’ 与 ‘Tab caption@icon’ 类似。

Flink

可在任何界面插入类似友情链接列表效果，内容格式与友情链接界面一样，支持 yml 格式。

ABCJS 乐谱

在页面上渲染乐谱，具体查看官方文档。

Series 系列文章

在文章的 front-matter 上添加参数 series，并给与一个标识。使用此标签外挂，会把相同标识的文章以列表的形式展示。如果不写 series 标识，则默认为你使用此标签外挂所在的文章的 series 标识。

chartId 【可选】图表 ID
